Finger Protocol TCP Port

TCP Port 79

by Charles Iliya Krempeaux (reiver)

The default TCP port for the finger protocol is 79.

This TCP Port (79) is what most finger protocol servers and finger protocol clients connect to.

Alternative TCP Port

The finger protocol seems to have had an alternative unofficial TCP port, other than TCP port 79:

TCP port 2003.

It was used by GNU cfinger.

TCP-port 2003 would not typically require 'root' privileges — in systems that requires 'root' privileges for TCP ports less-than 1024.
